# Railway Booking Data Analysis

## Overview
This project analyzes a railway booking database using SQL and Python to extract
insights related to passenger bookings, train occupancy, and booking status.

## Dataset
- Academic railway booking dataset stored in SQLite
- Tables include Passenger, Train, Booked, and Train_Status
- Data contains passenger details, booking status, train routes, and travel dates

## Tools Used
- SQL (Joins, Aggregations, Filters)
- Python
- SQLite
- Streamlit

## Key Analysis Performed
- Passenger count per train
- Bookings filtered by date
- Booking status analysis (Booked / Waitlisted)
- Age-based passenger segmentation

## Key Insights
- Certain trains show consistently higher passenger occupancy
- Booking status analysis helps understand demand patterns
- Aggregated passenger counts support operational planning

## Notes
This is an academic project completed as part of a graduate-level course using
non-confidential sample data.
